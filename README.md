# Osijek AI Guide

**AI-powered guide for the city of Osijek** — built with Grok API and RAG.

Helps tourists and locals with attractions, food, events, history, and daily life.

## Features

- Powered by Grok API (xAI)
- Retrieval-Augmented Generation (RAG)
- Answers in Croatian, English, and German
- Covers tourism, history, food, events, and practical information

## How to Run

```bash
pip install -r requirements.txt
streamlit run src/app.py
